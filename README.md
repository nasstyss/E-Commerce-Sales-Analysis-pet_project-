# E-Commerce-Sales-Analysis (pet project)
## О проекте
---
Проект посвящен анализу данных интернет-магазина.
Цель - выявить ключевые продуктовые категории, определить влияние цены на продажи и оценки пользователей, проанализировать сезонные тренды и оценить уровень удовлетворенности клиентов.

В проекте проведен полный цикл аналитики: от первичной обработки данных до построения визуализаций и формулировок выводов.

---
## Структура проекта
1. Предобработка данных
- проверка пропусков, дубликатов и выбросов
- приведение типов данных
2. Анализ продуктовых категорий
- создание дополнительных признаков (total_sales, revenue)
- определение категорий-лидеров по продажам и выручке
- сравнение различий между ними
3. Анализ влияния цены и отзывов
- проверка корреляций между ценой, продажами и оценками
- визуализация взаимосвязей (scatter plots, heatmap)
- оценка по ценовым сегментам
4. Сезонные тенденции
- анализ продаж по месяцам
- выявление сезонного пика
- сравнение активности категорий в разные месяцы
5. Оценка удовлетворенности клиентов
- анализ распределения рейтингов
- выявление зависимостей между количеством продаж и пользовательскими отзывами 
- сравнение среднего рейтинга и числа отзывов по категориям
---

## Инструменты и библиотеки

Python, Pandas, Matplotlib, Seaborn, NumPy
